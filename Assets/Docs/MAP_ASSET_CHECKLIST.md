# TEFO Map Asset Checklist

Project: TEFO
Engine: Unity 2D
Style: top-down pixel-art vehicle open world
Tile size: 16x16 px
Pixels Per Unit: 16
Target biomes: forest, rural village, urban town, beach/ocean

This document tracks the expected source PNG assets for the map system. Draw the
actual art later and place the files in the matching `Assets/Art/Map/...`
folders. Do not place generated Unity Tile assets in the source art folders.
